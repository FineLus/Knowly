# Knowly (큐엔에이 시스템)

---

# 프로젝트 개요

- 이 프로젝트는 사용자가 업로드한 텍스트 파일을 AI를 활용하여 요약하고, 해당 요약된 내용을 기반으로 질문에 답변하는 웹 애플리케이션입니다. 사용자는 PDF 파일을 업로드하고, 시스템은 파일 내의 텍스트를 요약한 후, 그 요약된 내용을 바탕으로 질문을 받을 수 있습니다. 이 프로젝트는 자연어 처리(NLP) 모델을 활용하여 사용자가 입력한 텍스트를 분석하고 요약 및 질의 응답을 제공합니다.

@주요 기능

- PDF 파일 업로드: 사용자가 PDF 파일을 업로드하여 텍스트를 추출합니다.

- 텍스트 요약: 추출된 텍스트를 AI 모델을 사용해 간결하게 요약합니다.

- 질문 응답: 요약된 텍스트를 바탕으로 사용자가 질문을 입력하고, 그에 맞는 답변을 제공합니다.

- 간단한 웹 인터페이스: Flask를 사용한 직관적인 웹 애플리케이션으로 파일 업로드, 요약 보기, 질문 입력 기능을 제공합니다.

# 기술 스택

- 프레임워크: Flask

- 언어: Python

- 자연어 처리 모델: Hugging Face Transformers (BART for summarization, RoBERTa for Q&A)

- 파일 처리: PyPDF2 (PDF에서 텍스트 추출)

- AI 모델 라이브러리: transformers, torch

- 기타: HTML, CSS (웹 인터페이스)

# 프로젝트 구조

        /templates        # HTML 템플릿 파일

# 보완하거나 성장할 부분

- 파일 형식 확장: 현재 PDF 파일만 지원되지만, 다른 파일 형식 (예: DOCX, TXT 등)도 처리할 수 있도록 확장할 수 있습니다.

- 모델 성능 개선: 텍스트가 매우 길거나 복잡할 경우 성능이 저하될 수 있으므로, 더 효율적인 요약 및 질문 응답 모델을 연구하여 성능을 개선할 수 있습니다.

- UI/UX 개선: 사용자 인터페이스가 매우 기본적이므로, 더 직관적이고 아름다운 UI로 개선할 수 있습니다.

- 파일 크기 처리: 큰 파일을 업로드할 때 성능 최적화가 필요하며, 파일 크기 제한을 설정하는 기능을 추가할 수 있습니다.

- 오류 처리 강화: 예외 처리와 오류 메시지를 사용자 친화적으로 개선하여, 시스템이 원활하게 동작하도록 할 수 있습니다.
